# 利用者マニュアル — カップル家計簿

## 1. はじめに
本アプリは、カップル間のお金のやり取り（割り勘・立て替え）を管理し、  
**「誰が誰にいくら支払えば清算できるか」** を即座に確認できる Web アプリです。

---

## 2. 画面構成

| 画面 | 説明 |
|------|------|
| ダッシュボード | 清算額・カテゴリ別支出・直近取引を表示 |
| 支出登録 | 新しい支出を登録するフォーム |
| 履歴一覧 | 全取引の閲覧・編集・削除 |

---

## 3. 支出を登録する

1. ナビゲーションバーの **「支出登録」** または画面右下の **＋ボタン** をタップ
2. 以下を入力:
   - **日付**: デフォルトは今日の日付
   - **品目**: 例）スーパーで買い物
   - **金額**: 1 円以上の整数
   - **支払った人**: UserA または UserB を選択
   - **カテゴリ**: 食費、光熱費、旅行 など
   - **支払いの種類**:
     - 🙋 **自分用** — 自分だけの支出（清算対象外）
     - 🤝 **相手用** — 相手が全額払うべき支出（相手への貸し）
     - ➗ **割り勘** — 2 人で半額ずつ
3. **「登録する」** ボタンを押す

> 💡 **前回選択した支払人は次回自動的に選択されます。**

---

## 4. 清算額を確認する

ダッシュボード画面のメインカードに表示されます:

- 🔵 **青色カード** → 「UserB が UserA に 〇〇円 支払う必要があります」
- 🔴 **赤色カード** → 「UserA が UserB に 〇〇円 支払う必要があります」
- 🟣 **紫色カード** → 「清算不要です 🎉」

### 計算ロジック
```
清算額 = (Aが相手用に払った額 + Aの割り勘支出の半額)
       - (Bが相手用に払った額 + Bの割り勘支出の半額)
```
※ 「自分用」の支出は清算対象外です。

---

## 5. 支出を編集・削除する

1. **「履歴」** 画面を開く
2. 各取引の右側にあるボタンを使用:
   - ✏️ **ペンアイコン** → 編集画面へ移動
   - 🗑️ **ゴミ箱アイコン** → 削除確認ダイアログが表示
3. 削除は確認後に実行されます（元に戻せません）

## 7. Tips
- **モバイル対応**: スマートフォンのブラウザで快適に使えます
- **データ永続化**: 入力したデータは SQLite に保存され、サーバー再起動後も維持されます
- **ブックマーク推奨**: ホーム画面にブックマークしておくとアプリのように使えます
