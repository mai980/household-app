# プロジェクトサマリ — カップル家計簿

## プロジェクト名
カップル家計簿 Web アプリケーション

## 目的
カップル間の支出（割り勘・立て替え・個人支出）を記録し、  
清算額をリアルタイムに表示することで、お金のやりとりを簡単に管理する。

## 開発期間
2026 年 2 月

## 技術構成
| レイヤー | 技術 |
|----------|------|
| バックエンド | Python 3.9+, Flask 3.1 |
| ORM | SQLAlchemy 2.0 |
| データベース | SQLite (ファイルベース) |
| フロントエンド | HTML5, CSS3, Bootstrap 5, JavaScript |
| デザイン | ダーク UI / グラスモーフィズム / モバイルファースト |
| デプロイ | Gunicorn / Render.com / Heroku 対応 |

## 主な機能
1. **ダッシュボード** — 清算額をカラー分けで即座に表示
2. **支出登録** — 日付・品目・金額・支払人・カテゴリ・種別を記録
3. **履歴管理** — 全取引の一覧・編集・削除（ページネーション付き）
4. **カテゴリ別集計** — カテゴリごとの支出を可視化
5. **月次サマリ** — 当月の支出合計と件数を表示
